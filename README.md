## React Component Template

### 更改模版
- `package.json`
> `@hocgin/component_name`=> `YOUR COMPONENT NAME`
> `repository`=> `YOUR Repository URL`

### 发布 npm 
```shell script
# 1. 登陆
npm login
# 2. 编译+发布
npm run deploy
```

### 使用组件
```shell script
# 引入插件
yarn add @hocgin/gin-xx:latest
# 在 jsx 中使用
import GinXx from '@hocgin/gin-xx'
```
